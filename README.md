This Chrome extension allows you to rename any browser tab to a custom title. It is useful for organizing tabs, maintaining privacy, studying, or personal preference. The interface is simple, lightweight, and works on all websites.

---

## Features

- Rename any tab instantly
- Keeps the custom name even after reload
- Simple and minimal UI
- Lightweight and fast
- Works on all websites

---

## How to Install (Developer Mode)

1. Download the ZIP from repo 
3. Extract the ZIP  
4. Open Chrome and go to:  
   `chrome://extensions/`
5. Enable Developer Mode (top right)  
6. Click **Load unpacked**  
7. Select the extracted folder

The extension will now appear in your Chrome toolbar.
